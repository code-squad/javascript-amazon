# Amazon Prime Project

## Demo
* [Demo](https://amz-prime.firebaseapp.com/)

## Intro
* [Amazon Prime](https://www.amazon.com/prime) 메인페이지를 구현하는 프로젝트

## 사용기술
* HTML
* CSS
  - Preprocessors
    - Sass
* JavaScript
* Package Manager
  - yarn
* Module Loader/Bundler
  - webpack
* Deploy
  - firebase

## 사용법
* 패키지 설치
```bash
npm install
yarn install
```
* 로컬 개발 버전 실행
```bash
npm run dev
yarn run dev
```
* 배포를 위한 빌드
```bash
npm run build
yarn run build
```
